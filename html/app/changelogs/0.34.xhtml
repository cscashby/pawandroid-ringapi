Changelog - Version 0.34

This is mainly a bugfix release.

Server

  • PAW can now use SSL as connection protocol.
    If HTTPS is selected a 1024 bit self signed RSA certificate is used. The certificate has to be imported into the web browser. With HTTPS enabled the server response might be a little bit slower.

    HTTP is still used as default to not confuse users.

Web App

  • HTTP/HTTPS settings habe been added to the Server Settings.
