Changelog - Version 0.92

This release contains:

  • Port Check
  • Russian Translation
  • WiFi Lock
  • URL Unicode Supports
  • New BeanShell Version

App

Port Check

The app now checks if the selected port is available and shows an appropriate message. In former versions the message was generic and did lead do some confusion.

Russion Translation

The app is now also available in Russian (thanks to Dmitry Shvidunov).

Server

WiFi Lock

WiFi should stay on and should not be turned off if device goes to sleep.

URL Unicode Supports

Unicode characters are now supported inside URLs.

New BeanShell Version

BeanShell has to be updated to BeanShell 2.1.8
