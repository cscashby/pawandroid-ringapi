Changelog - Version 0.61

This realease contains two new settings. Apart from that it's mostly a bugfix release.

Server

  • Support for Freeze Content setting: If the content is frozen no content extraction will be performed on new versions of PAW.
  • Support for Show URL in Notification setting: Shows the URL in the notification instead of the generic message.
  • Fixed error in CGI Handler.

Web Application

  • Setting for Freeze Content and Show URL in Notification have been added to the server settings.
  • Login bug fixed.
  • Display error in Toolbar Settings has been fixed.
