Changelog - Version 0.58

Server

  • On user request, the html/index.xhtml file is no longer overwritten.
  • Added permissions: READ_HISTORY_BOOKMARKS and WRITE_HISTORY_BOOKMARKS
  • App does not crash during orientation change on content extraction.
  • IPv6 display bug fixed (Motorola Xoom).
  • AndroidCgiHandler has been added, which enables CGI support. PHP will be available as plug-in.

Web App

  • SMS thread view does not break on drafts.
  • Plug-ins are now sorted.
  • The login page does not work with IE9. To (hopefully fix this) the following meta tag has been added to the login page:
    <meta http-equiv="X-UA-Compatible" content="IE=EmulateIE8" />
  • New Plug-ins section in "Add-Ons" menu.
