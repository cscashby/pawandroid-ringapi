Changelog - Version 0.54

  This release mainly contains SMS improvements.

Web App

  • New threaded SMS view.
  • Sending of SMS has been improved:
    • Success/Failure should now be reported correctly.
    • Messages are copied to outbox.
    • Large messages are supported.
    • Multiple recipients are supported.
  • APK Backup now only shows unprotected apps. This was a bug, protected apps can not be copied anyway.
  • Share Content now detects URL and provides an icon to open it in a new browser window.

Android Functions

Improvements in the following function (sie PAW Functions for more detail):
  • Methods setComeraId(id) and getComeraId() have been added to Camera Preview
  • Intents returned by getIntentResultFromBroadcast() and getIntentResultsFromBroadcast() now include the result code as extra.
  • The sendSMS() command has been improved.

Get and Post parameters are combined and multiple variables are stored into an ArrayList.
