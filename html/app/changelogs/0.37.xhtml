Changelog - Version 0.37

This is mainly an Eclair/Froyo compatibility realease.

Server

  • Some server bugfixes concerning shutdown.
  • Froyo no longer supports the restarting of packages, so the server shutdown has been changed.
    This also affects the process list in the web app.

Web App

  • A Froyo info message has been added to the process list application that stopping processes might not work as before, because restarting packages is no longer possible.
  • READ_LOG permission has been added to the manifest, so logcat should now work again. This was broken for devices running Eclair.
