Changelog - Version 0.31

Server

  • Server settings now supports to change the maximum file upload size. This can be tweaked until a certain limit. Because uploaded data is kept in memory, Android process memory restrictions apply and so the maximum upload size might not be very large. The standard is 2 Mbytes.

Web App

  • New File Manager. Basically this is the file manager from Core Five Labs (http://labs.corefive.com/Projects/FileManager/) but slightly enhanced.
  • SMS messages now can also be sent to an individual number.
