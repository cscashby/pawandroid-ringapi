Changelog - Version 0.51

Server

  • No restriction on upload file sizes!
    The underlying Brazil framework has been modified, file size is only restricted by the browser and the filesystem size on the Android device. Standard upload size is still 2 MB, but can be increased to any size in the server settings.
  • Bluetooth permission has been added. An example how to use Bluetooth is available in the code snippet section.

Web App

  • Image Slideshow has been added to the Media menu.
