BeanShell

Include BeanShell XHTML Content

The BeanShell command include(filename) can be used to include text files containing XHTML BeanShell tags (<bsh> ... </bsh>).
The filenames can be absolute or relative to the current script location.

Example: 
<bsh>include("filename.inc");</bsh>

PAW Functions

File Upload

File uploads have now been simplified by providing the two uitlity methods UploadUtil.processUpload(File outDir, Request request, boolean overwrite) and UploadUtil.processUpload(File outDir, Request request, boolean overwrite, String[] allowedExtensions) which are available inside the de.fun2code.android.pawserver.util package.

The parameter outDir specifies the directory where to store the files. The second parameter is the request object available from within any xhtml page. The last parameter ovewrite specifies if files should be overwritten or not. If this parameter is false a number will be appended if a file already exists (e.g. foo-3.png).
In addition to the first method, the second has an additional parameter called allowedExtensions. This is a string arraw that specifies the allowed file extensions (case is ignored).
