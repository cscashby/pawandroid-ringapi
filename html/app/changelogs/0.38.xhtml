Changelog - Version 0.38

This release contains some fixes, fun stuff and a new add-on.

Web App

  • Find Mobile now sets back the original volume back correctly.
  • Voice Recorder added.
  • Sliding Image Puzzle added.
Both Voice Recorder and Sliding Image Puzzle were originally ment to be tutorials but due to the lack of time they are now in the standard app.
So no tutorials for the time being.

Add-Ons

This is new and is also represented by a new menu entry.
First add-on is a Greasemonkey script that allows to open market links from a web page directly on the Android device.

For more details see the add-on description.
